Blag

2011-08-10 | in meta

Finally, a simple blog system implemented using wok. There's not much to this yet, just a simple list of blog entries, but there are more features to come soon including:

  • Tag filtering so the user can chose to view only the entries associated with a particular tag.
  • Paging so all entries don't load on a single page.
  • JavaScript-based search (maybe)

If you want to see my current work-in-progress, you can find it on my development site. Source code available on GitHub.
